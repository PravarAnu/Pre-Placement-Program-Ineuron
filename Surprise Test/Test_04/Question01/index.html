The sticky property is used as a positioning for this box and you can see that this box is sticked on the top when you scroll the screen. You can also see the code below

position: sticky; background-color: rgb(3, 87, 59);
top:0px;

This is the default state (i.e static) of css positioning and the box is positioned according to the normal flow. The top, right, bottom, left and z-index properties have no effect.

position: static; background-color: rgb(255, 0, 0);
margin-left: 200px;

This is fixed state of this box where the box is removed from the normal document flow, and no space is created for the box in the page layout. Using this positioning you can fix the position of any element.

position: fixed; background-color: rgb(98, 98, 3);
left: 400px;

The element is positioned according to the normal flow of the document, and then offset relative to itself based on the values of top, right, bottom, and left.

position: relative; background-color: rgb(3, 80, 3);
top:200px;
left: 600px;

The element is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned relative to its closest positioned ancestor, if any; otherwise, it is placed relative to the initial containing block. Its final position is determined by the values of top, right, bottom, and left.

position: absolute; background-color: rgb(165, 42, 42);
top: 800px;
left: 800px
