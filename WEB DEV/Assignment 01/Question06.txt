Q. What are some of the advantages of HTML5 over its previous versions?

Ans-
The HTML5 introduces many advantages over previous versions that are as follows :- 
Multimedia Support - Html5 provides native support for audio and video elements.
Semantic Elements - Html5 introduced semantic elements to make web pages more accessible.
Canvas - Canvas is an element which is used for rendering graphics and animation directly in the browser without any need of plugin.
Local Storage - Html5 introduced the ability to store the data locally on the users device by using technologies like webstorage and indexedDB.
Geolocation - Html5  includes a geolocation API that allows web applications to retrieve the user location with their consent.
Responsive Design - Html5 introduced responsive design principles, making it easier to create websites, webpages that adapt different screen sizes. For example - <meta>, media query, touch events…
Improved From Controls - Html5 introduced several new form elements and attributes to enhance the user experience and provide better input validation such as <input type=”date”>, <input type=”email”>.
Cross Platform Compatibility - Html5 is supported by modern web browsers across different platforms, including desktop, laptop, mobile, tablet.
